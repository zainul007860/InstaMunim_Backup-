# InstaMunim POS Mobile App
"Beyond Billing" - A premium, high-performance POS solution for modern merchants.

## Features
- Real-time Inventory & Sales
- WhatsApp Receipts
- AI Insights
- Offline-ready
- Multi-theme (Day/Night Mode)

## Tech Stack
- Next.js (React)
- Tailwind CSS
- Supabase (Database)
- Lucide Icons

## Deployment
This app is ready to be deployed on **Vercel**. 🚀
